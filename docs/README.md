## 🎓 学习路径

### 路径 1: 快速了解项目 (30 分钟)

```
1. 项目总览 (10 分钟)
   └─ 了解项目是什么、能做什么
   
2. README 快速开始 (10 分钟)
   └─ 运行起来，体验功能
   
3. RAG 功能文档 - 概述 (10 分钟)
   └─ 理解核心功能
```

### 路径 2: 成为开发者 (2-3 小时)

```
1. 开发指南 - 环境搭建 (30 分钟)
   └─ 配置开发环境
   
2. 系统架构 (30 分钟)
   └─ 理解整体架构
   
3. 设计模式 - 核心章节 (60 分钟)
   └─ 深入代码结构
   
4. RAG 功能文档 (30 分钟)
   └─ 掌握核心功能
   
5. 开发指南 - 开发流程 (30 分钟)
   └─ 学习如何开发
```

### 路径 3: 部署到生产环境 (1-2 小时)

```
1. 部署指南 - Docker 部署 (30 分钟)
   └─ 快速部署
   
2. 部署指南 - 生产环境配置 (30 分钟)
   └─ 生产环境优化
   
3. 运维手册 (20 分钟)
   └─ 日常运维
   
4. 发布清单 (10 分钟)
   └─ 发布检查
```
