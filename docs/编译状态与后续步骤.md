## 🔍 具体错误列表

### 错误 1: SmartChunkingService - chunkByTime

**位置**: 第 227 行  
**问题**: 返回类型应为 `List<Chunk>` 但创建了 `TravelKnowledgeSnippet`  
**修复方案**: 
```java
// 当前 (错误)
private List<Chunk> chunkByTime(TravelKnowledgeSnippet snippet) {
    List<TravelKnowledgeSnippet> chunks = new ArrayList<>();  // ❌
    chunks.add(new TravelKnowledgeSnippet(...));  // ❌
}

// 应改为
private List<Chunk> chunkByTime(TravelKnowledgeSnippet snippet) {
    List<Chunk> chunks = new ArrayList<>();  // ✅
    chunks.add(new Chunk(season, chunkContent));  // ✅
}
```

### 错误 2: SmartChunkingService - chunkWithOverlap

**位置**: 第 300 行  
**问题**: 同 chunkByTime  
**修复方案**: 同上

### 错误 3: RerankingService - Lambda 变量

**位置**: 第 410 行  
**问题**: Lambda 表达式引用的变量必须是 final  
**修复方案**:
```java
// 当前 (错误)
double diversityWeight = 0.2;
snippets.forEach(s -> {
    score -= diversityWeight * similarity;  // ❌ diversityWeight 需要是 final
});

// 应改为
final double diversityWeight = 0.2;  // ✅ 添加 final
snippets.forEach(s -> {
    score -= diversityWeight * similarity;
});
```
